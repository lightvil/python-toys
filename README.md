# python-toys
* MarketBuddy 개발 과정에서 필요한 Python 코드를 테스트하고 개발하기 위한 프로젝트
* 사용할 것들
    * Anaconda
    * Python 3.10
    * Docker
    * VSCode

# 환경 준비
1. Anaconda, VSCode 설치
    * VSCode Extension들:
1. Anaconda Env생성
   ```
   conda create --name python-toys python=3.10
   conda activate python-toyes
   ```
1. 기본 패키지 설치
    ```
    # MarketBuddy의 requirements
    pip install -r requirements.txt
    ```
